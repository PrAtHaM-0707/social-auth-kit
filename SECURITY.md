# Security Policy

## Supported Versions

Only the latest major version is highly supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within `social-auth-kit`, please send an e-mail or create a private GitHub advisory. All security vulnerabilities will be promptly addressed.

Please do not publicly disclose the issue until it has been addressed by the maintainers.
